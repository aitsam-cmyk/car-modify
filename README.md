# Car Garage (local)

Instructions:
1. Install Node.js (LTS) which includes npm.
2. Install deps:
   npm install
3. Start server:
   npm start
4. Open in browser:
   http://localhost:3000
5. To expose public URL use ngrok or localtunnel (see EXPOSE.md).

Repo notes:
- Do not commit node_modules or the SQLite DB (cargarage.db); they are in .gitignore.
